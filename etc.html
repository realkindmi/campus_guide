MO 만족도조사

만족도조사 페이지 작업시 작업 순서

- 개발팀에서 1차 개발 작업 후 퍼블리싱 작업자에게 폴더 및 경로 전달

- 만족도조사 페이지 초기화 URL 및 권한계정 전달 받기   ex) 강남 / 562883

- 만족도조사 테스트 페이지 계정 전달 받기   ex) mstest001 / @mega7788

만족도조사 CSS 작업

  • 공통 CSS 위치/common/bootstrap_4.1.2/min/survey.css

- 작업 시 Class명을 새로 생성하여 추가하는 방식으로 작업   ex) .perpl20230424

- 모바일 디바이스별 해상도 작업은 width : 320px 까지

인트로 공지사항(PC/MO)

공지사항 페이지 작업시 작업 순서

- HTML으로 먼저 퍼블리싱 작업

- Admin 실서버 사이트 > 학원관리게시판 > 공지사항에서 작업물 반영

- 메모장 등에 한 번 Copy&Paste 한 뒤에, Admin 공지사항에 소스를 붙여 넣는다. (바로 붙여넣을 시, 알 수 없는 태그가 붙는 경우 있음)

공지사항 HTML 작업경로 예시

  • PC https://tcampus.megastudy.net/test/notice_240401.html
  • MO https://tmcampus.megastudy.net/test/notice_240402.html
  • PC Admin 반영 https://campus.megastudy.net/common/notice/notice_view.asp?code=23806
  • MO Admin 반영 https://mcampus.megastudy.net/common/notice/notice_view.asp?code=23806
